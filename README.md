# larabbs

## 简介

## 安装
```sh
cd your-project/
git clone https://github.com/lePig/larabbs.git larabbs
```

## 使用
```sh
cd larabbs
composer install
```
建立一个`larabbs`数据库
```sql
CREATE DATABASE `larabbs`;
```

## 访问
* 配置本地`hosts`文件, windows下为`C:\Windows\System32\drivers\etc\hosts` linux下为`/etc/hosts`
* 浏览器键入`http://larabbs.test`查看效果。
